EXAMPLE

HOW TO PLAY

1.การ run ตัวเกมจะต้องมีการ compiler ผ่านตัว gcc ใน cygwin terminal เมื่อทำการ compiler เสร็จสิ้น ก็ทำการ run ไฟล์ที่ compiler แล้วได้เลย ในรูปชื่อไฟล์ที่ compiler แล้วคือ run.exe

2.เมื่อ run เข้ามาก็จะพบกับหน้าตัวอย่างการ input หรือการเล่นเกมนี้นั่นเอง จะมีวิธีการเล่นและคำเตือนบอกไว้อย่างชัดเจน

3.การเล่นขั้นแรกก็คือจะมีตัวอักษร X กระพริบที่ช่องใดช่องหนึ่งแบบสุ่มให้เราจำให้ดีว่าช่องนั้นคือช่องไหน

4.หลักจากที่ตัวอักษร X กระพริบเสร็จตัวเลขก็จะขึ้นตามช่องดังภาพ ให้ผู้เล่นกรอกตัวเลขที่ขึ้นในช่องที่เคยมีตัวอักษร X กระพริบลงในบรรทัดสำหรับการตอบเมื่อตอบเสร็จถ้าถูกต้องก็จะขึ้น Correct แล้วก็จะไปใน round ต่อๆไป แต่ถ้าผิดก็จะขึ้น Incorrect แล้วก็จะย้อนกลับมาเล่นใน round นั้นอีกรอบ

5.เมื่อเราเล่นจนจบก็จะขึ้นข้อความว่า You win!!! ในบรรทัดต่อมาก็จะถามผู้เล่นว่าต้องการปิดเกมนี้เลยไหมถ้าพิมพ์ y ก็คือการออกจากเกม ถ้าพิมพ์ n เกมก็จะทำการเริ่มต้นใหม่อีกครั้ง

// ฟังชั่น delay

void delay(int number_of_seconds)
{
// Converting time into milli_seconds
int milli_seconds = 1000 * number_of_seconds;

// Storing start time
clock_t start_time = clock();

// looping till required time is not achieved
while (clock() < start_time + milli_seconds)
;
}
